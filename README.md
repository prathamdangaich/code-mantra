# Code Mantra  

**Code Mantra** is a real-time collaborative code editor that enables multiple users to work together on the same code, in the same editor, at the same time.  
It‚Äôs designed for **doubt resolution**, **pair programming**, **interview preparation**, and **team coding sessions**.

## Features  

- **Real-time Code Synchronization** ‚Äì Any code changes are instantly reflected for all users in the same room.
- **Compiler support** - Anyone inside the same room and compile the code.
- **Multiple Language Support** ‚Äì Switch between JavaScript, Python, C++, and Java.
- **Room-based Collaboration** ‚Äì Join or create a room using a unique Room ID.
- **User Presence Tracking** ‚Äì See who is in the room in real time.
- **Typing Indicator** ‚Äì Know when someone is typing.
- **Clipboard Copy** ‚Äì Quickly copy the Room ID to share with others.
- **Leave Room Safely** ‚Äì Automatically handle room exit and user updates.

## How to Use

1. Open the app in your browser.  
2. Enter a **Room ID** and your **Name**, then click **Join Room**.  
3. Share the Room ID with your friends/teammates.  
4. Start coding together in real time!
